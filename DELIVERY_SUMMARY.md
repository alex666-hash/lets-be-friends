## 🚀 Quick Start (3 Steps)

### Step 1: Backend Setup
```bash
cd backend
npm install
npm run dev
```
Server runs on `http://localhost:5000`

### Step 2: Frontend Setup (New Terminal)
```bash
cd frontend
npm install
npm run dev
```
App runs on `http://localhost:5173`

### Step 3: Configure Environment
Create `backend/.env` with your credentials:
```env
MONGODB_URI=mongodb://localhost:27017/lets-be-friends
JWT_SECRET=your_secret_key
GOOGLE_CLIENT_ID=your_google_id
FACEBOOK_APP_ID=your_fb_id
... (see INSTALLATION.md for complete list)
```

**That's it! Open http://localhost:5173 and start!**

Compatibility calculated as:
```

         Verification(10) + ProfileQuality(5) = 0-100
```
